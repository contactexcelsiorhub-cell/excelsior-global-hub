# Excelsior Global Hub Website

## Overview
This is the official homepage for **Excelsior Global Hub**, built with brand colors **Blue (#002f6c)** and **White (#ffffff)**.  
It highlights our ventures, mission, and impact across Fiji, the Pacific, and the world.

## Structure
- `index.html` → Main webpage structure
- `styles.css` → Styling with brand colors and responsive layout
- `script.js` → Basic interactivity for venture categories
- `README.md` → Documentation and project overview

## Features
- Clean, professional design in brand colors
- Responsive venture section with hover effects
- Interactive venture selection alerts
- Formal intro tagline:  
  *Connecting Fiji, the Pacific, and the World Through Smart Tech, Compliance, Responsible Innovation, Brand Partnerships, Strategic Marketing, and Sustainable Practices That Create Real Impact.*

## How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
3. Customize `styles.css` and `script.js` as needed.
